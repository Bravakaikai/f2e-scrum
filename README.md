# 系統說明

專案運行方式

### 1. Install dependencies
```
$ npm install
```

### 2. Compiles
```
$ npm run dev
```

# 資料夾說明

- src/assets - 圖片及資料放置處
- src/components - 元件放置處
- src/layouts - 共用版型放置處
- src/router - 路徑
- src/views - 頁面放置處
- src/store - 共用資料放置處

# 使用技術

- Vue3 Composition API 
- Vite
